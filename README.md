# Habit Tracker

A simple Habit Tracker built with Angular to practice components, state management, and forms.  
This project is a first step in building real front-end applications and deploying them online.

---

## Features

- Add new habits
- Mark habits as completed
- View a list of habits
- Toggle completion status

---

## Technologies

- Angular 21
- TypeScript
- HTML / CSS

---

## Development Server

To start a local development server, run:

ng serve
